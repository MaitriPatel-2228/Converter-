## 📌 Features

- 🔁 Convert between multiple units of **length** (meter, kilometer, mile, inch, etc.)
- ⚖️ Convert between **weight units** (gram, kilogram, pound, ounce, etc.)
- ⏱️ Convert between **time units** (seconds, minutes, hours, days, etc.)
- ⚡ Instant and accurate conversion results
- 📱 Responsive design for mobile and desktop
- 🧮 Simple and user-friendly interface

## 🛠️ Built With

- HTML5
- CSS
- JavaScript
